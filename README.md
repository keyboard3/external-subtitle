# external-subtitle
外挂字幕

## 功能特点
1. 支持将字幕嵌入到各种视频网站中
2. 支持抓取 YouTube CC 字幕并智能分句
3. 支持抓取原视频字幕
4. 支持本地视频字幕提取功能
5. 提供了便捷的字幕编辑与修改功能。
6. 提供了谷歌翻译字幕的功能

## 安装

- [Chrome](https://chromewebstore.google.com/detail/external-subtitle/peiaajeeocehgpbdbljhjenlikfkokdm?hl=zh-CN&utm_source=ext_sidebar)
